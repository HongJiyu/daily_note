# 第三章 加载模块

1.取消了js的全局名称空间，而是采用CommonJS模块系统取代。

2.导出时使用module.exports 和exports的关系。

3.导入时：

- 核心模块、npm i的模块，用模块名
- 绝对路径、相对路径（./），可以省略.js
- 文件夹，默认找文件夹下的package.json中的main属性的值，再拼接地址。否则找index.js 文件
- 从文件所在路径找 ./node_moudules/xxx 再文件的上一级目录： ../node_modules/xxx

4.导入模块时，被导的模块只会被初始化一次。
